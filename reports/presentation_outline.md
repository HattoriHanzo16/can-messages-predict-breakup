# Presentation Outline (10-15 min)

## 1) Introduction (2-3 min)
- Problem statement: Can messages predict breakup-related outcomes?
- Why it matters (insights for relationship counseling and communication patterns)
- Dataset overview and labeling strategy

## 2) Methodology (3-4 min)
- Data preprocessing steps
- Feature engineering (text length, word count, TF-IDF)
- Model selection: Logistic Regression vs Decision Tree

## 3) Live Demo (4-5 min)
- Show EDA visualizations
- Run model notebook and compare metrics

## 4) Results & Conclusions (2-3 min)
- Compare performance metrics
- Discuss limitations (proxy labels, subreddit bias)
- Future work (more robust labels, sentiment features, time-based analysis)
